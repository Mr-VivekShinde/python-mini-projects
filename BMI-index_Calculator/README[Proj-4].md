Q. What is BMI? 
ans: Body Mass Index (BMI) is a measure of body fat based on an individual's weight and height
   
   
   BMI= (weight/(height)**2)   kg/sq.m           

 
   Categories          BMI-index(kg/sq.m)
1) Underweight         <= 18.5
2) Normal weight       18.5-24.9
3) Overweight          25.0-29.9
4) Obese               30.0-39.9
5) Morbidly OBESE      >= 40
